# Amit Hair Studio Landing Page

A modern, single-page website for Amit Hair Studio with service highlights, gallery, pricing, testimonials, and direct booking via WhatsApp.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Bootstrap Icons (CDN)

## Features

- Responsive navigation with working mobile hamburger menu
- WhatsApp quick booking links across key sections
- Service, gallery, testimonial, and pricing sections
- Contact section with studio information and social links
- Scroll reveal animations for section content

## Project Structure

- `index.html` - Main page structure and lightweight interaction scripts
- `style.css` - Complete visual styling and responsive behavior

## Run Locally

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` directly in a browser.

Optional (recommended for local development):

- Use VS Code Live Server extension and serve `index.html`.

## Notes

- Bootstrap Icons are loaded via CDN in `index.html`.
- External images are served from Unsplash URLs.
